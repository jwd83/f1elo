# Team Logo Sources

These SVGs are bundled for local static display in constructor badges.
Most are from Simple Icons, which is published under CC0 1.0.
Some Formula One team-specific text logos are from Wikimedia Commons pages that
mark the files as public domain text logos. Logos may still be trademarks.
